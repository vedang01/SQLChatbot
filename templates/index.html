Welcome to SQL Wizard

SQL Wizard is a personalized educational chatbot designed to teach SQL. There are several features you can use to enhance your SQL learning journey.

Once logged in, you can:

  • Select a Topic: Choose a specific topic to learn from a dropdown, or simply select one of the suggested topics.
  • Suggested Topics: New topics are suggested based on how confident you are in your completed topics.
  • SQL Wizard Chatbot: Use the chatbot to solve any SQL-related queries.
  • SQL Overview: Get a beginner-friendly overview of SQL concepts and usage.
  • Sandbox Mode: Practice writing SQL queries against a live database.
  • Progress Tracker: View your learning progress and review topics you've mastered.
